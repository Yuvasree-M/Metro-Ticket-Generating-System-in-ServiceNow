# Metro-Ticket-Generating-System-in-ServiceNow

Overview

The Metro Ticket Generating System is a ServiceNow-based web application that automates metro ticket booking. Users can select source and destination stations, generate tickets digitally, and view fare details, while administrators manage stations, fares, and reports.

Features

  Digital ticket booking via Service Portal
  
  Automatic fare calculation
  
  Unique ticket ID generation
  
  Centralized ticket database
  
  Admin dashboards and reports

Modules

  Ticket Booking
  
  Ticket Management
  
  Admin Management
  
  Reports & Dashboards

Technology Stack

  Platform: ServiceNow
  
  Frontend: Service Portal
  
  Backend: Flow Designer, Business Rules
  
  Database: ServiceNow Custom Tables
  
  Reporting: Reports & Dashboards

Conclusion

This project demonstrates how ServiceNow can be used to build real-world business applications beyond ITSM, providing hands-on experience in platform development and automation.
